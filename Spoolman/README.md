# Spoolman Collections

Postman collection for Spoolman REST API v1.

## What’s included
- `Spoolman REST API v1.postman_collection.json`

## Details
- API: Spoolman REST API v1 for spool management workflows
- Use cases: spool tracking, lifecycle operations, and inventory control
- Authentication: configure as required by Spoolman API (API key / bearer token)

## Quick start
1. Import `Spoolman REST API v1.postman_collection.json` into Postman.
2. Create an environment or global variables for auth credentials.
3. Set base URL to your Spoolman API endpoint.
4. Execute requests for spool list, spool details, and status updates.

## Notes
- Adapt headers and body payloads to your deployment specifics.
- Use proper API credentials and secure token storage.
