# Xiaomi Collections

Postman collections for Xiaomi Cloud API to manage IoT devices and smart home interactions.

## What’s included
- `Xiaomi_Cloud_API.postman_collection.json`
- `Xiaomi_Cloud_API-1.postman_collection.json`

## Details
- Device management and control
- Cloud monitoring and status queries
- Support for primary and extended endpoints

## Quick start
1. Import one or both collection files.
2. Configure auth token and API keys according to Xiaomi Cloud guidelines.
3. Run sample device list and control calls.

## Notes
- Xiaomi Cloud APIs require a verified Xiaomi developer account.
- Token/credentials are sensitive; keep them secure in environment variables.
