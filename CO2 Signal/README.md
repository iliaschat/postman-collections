# CO2 Signal Collections

API collection for real-time carbon intensity data per country from CO2 Signal.

## What’s included
- `Routes.postman_collection.json`
- `CO2 Signal Environment.postman_environment.json`

## Details
- Endpoint: Get carbon intensity by country
- Auth: API key via `auth-token` header
- Usage: query grid emissions and embed into sustainability tracking

## Quick start
1. Sign up for CO2 Signal and get an API key.
2. Import collection and environment into Postman.
3. Set `CountryCode` and `auth-token` in environment.
4. Run `Get latest by country code`.

## Helpful notes
- Use ISO country code (e.g., `DE`, `US`, `FR`).
- Rate limits are set by CO2 Signal API plan.
