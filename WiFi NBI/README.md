# WiFi NBI Collections

Network-Based Interface (NBI) Postman collection for WiFi systems (e.g., Ruckus SmartZone).

## What’s included
- `Wifi NBI.postman_collection.json`

## Details
- Synchronous login endpoints
- User session management and logout
- Captive portal interactions

## Quick start
1. Import `Wifi NBI.postman_collection.json`.
2. Configure base URL and session variables in Postman, if required by your vendor.
3. Run the login request and pursue session flow tests.

## Notes
- Endpoint names and attributes can vary by provider (Ruckus, other controllers).
- Credentials and shielded network data should be managed securely.
