# Weather Collections

Postman collections for general weather APIs and Meteo Server in the Netherlands.

## What’s included
- `Weather.postman_collection.json`
- `Meteo Server.postman_collection.json`
- `Weather.postman_environment.json`

## Details
- `Weather`: generic location-based weather endpoints
- `Meteo Server`: Dutch meteorological data by GPS coordinates
- Environment variables include `latitude`, `longitude`, `METEOSERVER-URL`, `METEOSERVER-APIKEY`

## Quick start
1. Import one or both collections.
2. Import `Weather.postman_environment.json`.
3. Set coordinates and API key values.
4. Run requests (current conditions, forecast, synoptic data).

## Notes
- Use local coordinates for the precise forecast.
- The Meteo Server key may require registration.
